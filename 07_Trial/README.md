# 07_Trial

Trial preparation workspace.

## Anticipated documents

- `exhibit_list.md` defendant's exhibit list with TRE 901 authentication path per exhibit
- `witness_list.md` defendant's witnesses with anticipated testimony summary
- `cross_outlines/` per-witness cross outlines (Seat 03)
- `voir_dire.md` if jury demanded under TRCP 504.1
- `motion_in_limine.md` package (Seat 08 leads on Tex. Transp. Code § 550.065 exclusion)
- `trial_brief.md` short brief on Texas authority for the judge
- `closing_outline.md` map every cross hit to a burden-of-proof gap
- `preservation_log.md` running record of every objection and adverse ruling for de novo retrial in County Court (Seat 07)

## Owners

Seat 01 Lead Counsel (theory and synthesis); Seat 03 Cross-Examination & Trial Tactics; Seat 04 Justice Court Procedure; Seat 08 Evidence; Seat 09 Damages; Seat 07 Appeals (preservation).

## Texas authority quick reference

- TRCP 504 (J.P. Court trial procedure)
- TRCP 504.1 (jury, six members)
- TRE 401, 403, 611, 615
- Tex. Transp. Code § 550.065 (police report bar)
